Arduino LED Control (Pin 13)

این پروژه یک مثال ساده و پایه‌ای از کنترل خروجی دیجیتال در Arduino Uno است. در این برنامه، LED داخلی برد که روی پین 13 قرار دارد، همیشه روشن نگه داشته می‌شود.

🔧 سخت‌افزار مورد نیاز


آردوینو Uno
کابل USB
نیاز به led نیست (led)داخلی
🧩 کد
int led = 13;

void setup() {
  pinMode(led, OUTPUT);
}

void loop() {
  digitalWrite(led, HIGH);
}

🚀 نحوه اجرا

پروژه را در Arduino IDE باز کنید.

برد Arduino Uno و پورت مربوطه را انتخاب کنید.

فایل را آپلود کنید.



پین ۱۳ به‌عنوان خروجی تنظیم شده است.

در حلقه اصلی، مقدار HIGH باعث روشن شدن LED داخلی می‌شود.
