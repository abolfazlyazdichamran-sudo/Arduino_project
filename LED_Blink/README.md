پروژه روشن کردن LED با Arduino UNO

این پروژه یک LED را که به پایه ۱۳ آردوینو متصل است روشن نگه می‌دارد.
این تمرین یکی از پایه‌ای‌ترین پروژه‌ها برای یادگیری خروجی‌های دیجیتال در آردوینو است.
برد Arduino UNO به‌صورت پیش‌فرض دارای یک LED داخلی روی پایه ۱۳ است، بنابراین اجرای این پروژه بدون نیاز به LED خارجی نیز ممکن است.
سخت‌افزار مورد نیاز

برد Arduino UNO

(اختیاری) LED

(اختیاری) مقاومت ۲۲۰ اهم

کابل USB برای آپلود کد
نحوه اتصال (در صورت استفاده از LED خارجی)
پایه 13 آردوینو  →  مقاومت 220 اهم  →  پایه مثبت LED
پایه GND آردوینو →  پایه منفی LED

توضیح عملکرد

در بخش setup پایه ۱۳ به‌عنوان خروجی تنظیم می‌شود.

در بخش loop دستور digitalWrite(HIGH) باعث می‌شود پایه همیشه مقدار ۵ ولت تولید کند.

در نتیجه LED همیشه روشن باقی می‌ماند.
